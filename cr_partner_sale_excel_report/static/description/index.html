  • Navigate/Open any Contact let's take Abigail Peterson in these case.
  • A wizard named "Sale Report" is added in action menu in Customer model.

  • A wizard layout contains Start Date and End Date fields and 2 buttons PDF Report & Excel Report.
  • Here user can set Start Date and End Date and when user click on Excel Report or PDF Report button user will have their excel report or pdf report downloaded.
